tomcat安装

发表于 2017-08-18 | 分类于 java

1.Tomcat介绍

  • Web服务器与应用服务器的概念
    web服务器：专门处理HTTP请求，提供WEB信息的浏览服务器
    应用服务器：可解析运行后台代码为应用程序提供商业逻辑
  • Tomcat与Servlet
    Servlet 是一种运行在Java语言的服务器上的组件
    Tomcat是一个Servle容器，也是具备WEB服务器能力的轻量级
  • Tocat 的主页
    http://tomcat.apache.org/index.html

2.在Windows上安装Tomcat

  • 在Windows上安装Java环境
    下载jdk Downloads->Java SE Downloads->Windows x64(jdk包含jre)
    1.jdk目录下有各种工具。
    2.配置环境变量。
    选中桌面计算机属性-高级系统设置-环境变量（用户变量对当前用户,系统变量对所有用户有效，）
    3.新建变量-变量名（PATH）-变量值（jdk安装目录）
    http://www.oracle.com/technetwork/java/javase/downloads/index.html
  • 在Windows 上安装Tomcat（分.exe 和 .zip）
    .exe安装先安装好JDK默认一直点击下一步
    .zip安装必须配置JAVA_HOME，解压到指定目录(
    1.配置JAVA_HOME（变量名：JAVA_HOME，变量值：jdk安装目录
    ）
    2.编辑PATH，（变量名：PATH 变量值：%JAVA_HOME%\bin）；

3.在Linux上安装Tomcat

1.下载后缀为tar.gz JDK 和 Tomcat
2.上传到Linux
如:scp root@10.211.55.5:~/soapui-settings.xml soapui-settings.xml1
scp soapui-settings.xml root@10.211.55.5:~
3.安装JDK
1.创建目录# mkdir /user/local/java
2.解压# tar -zxvf jdk-8u121-linux-x64.tar.gz -C /user/local/java/
4.配置环境变量# vim /etc/profile
JAVA_HOME=/user/local/java/jdk1.8.0_121
PATH=$JAVA_HOME:$PATH
export JAVA_HOME PATH
source /etc/profile (使配置生效)
java -version （查看版本）
5.安装Tomcat
1.解压# tar -zxvf apache-tomcat-9.0.0.M17.tar.gz -C /user/local/
2.启动tomcat# sh startup.sh
6.如何无法访问Tomcat
关闭一下防火墙# service iptables stop 试试
7.产Tomcat进程# ps -ef |grep tomcat 关闭# kill -9 5144

4.Tomcat目录结构

  • bin 存放启动和停止Tomcat的脚本文件
  • conf 存放Tomcat所有的配置信息的文件
  • lib 存放Tomcat所有的功能活依赖的jar文件
  • logs 存放Tomcat 运行时的日志文件
  • temp 存放Tomcat运行时所产生的临时文件
  • work 处理jsp页面的工作目录
  • webspp Tomcat 维护的所有的web应用程序所在的目录
